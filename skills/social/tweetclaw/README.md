# TweetClaw

TweetClaw is the maintained Xquik OpenClaw plugin for X/Twitter agent workflows.
It helps agents search tweets, scrape tweet data, post tweets and replies,
monitor X/Twitter activity, inspect followers, upload or download media, manage
direct messages, and run giveaway draws through structured Xquik endpoints.
